The scope of this script is to generate the f_list file automatically from a given path.

Prerequisite:
Install python 3.2 or higher version

Input:
User has to provide the path of logs

Output:
flist.txt file will be generated once the scipt is executed with input argument

Procedure:
	-i Open the command prompt where the python file(flist_gen.py) available
	-i Type python flist_gen.py inputpath in command and run it
	-o f_list.txt will be generated in the same python script path

Verification
	- Check if the file exit with mf4 file or not
