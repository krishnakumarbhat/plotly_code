3. Device Driver Installation, Update and Uninstall
---------------------------------------------------

For updates please check the Vector internet homepage http://www.vector.com/latest_driver

All Installation and Uninstallation actions should be done with the Vector Driver Setup
(Setup.exe located in the download package or the drivers folder on the Vector Driver Disk).
For all operations Administrator rights are required.

For additional help and information please refer to the installation manual shipped with the 
driver CD or available on the Vector internet homepage (http://www.vector.com, section 
support). 

For troubleshooting please refer to the installation manual of the corresponding device.

--------------------------------------------------------------------------------------------

4. Vector Hardware Configuration utility (vHardwareManager.exe) 
---------------------------------------------------------------

The Vector Hardware Manager (vHardwareManager.exe) gives you the ability to configurate all
installed Vector Hardware. You will find the Vector Hardware Manager and the according help
in the installation directory:
C:\Program Files (x86)\Vector Hardware Manager\vHardwareManager.exe

5. Support 
----------

If you need technical support, you find additional information on the Vector internet 
homepage (http://www.vector.com) or contact Vector by mail (see mail addresses listed below) 
and send the following information: 
    - logfile generated by the Vector Hardware Configuration Utility 
    - version number of the driver
    - version number of the device's firmware  
    - serial number of the hardware
    - application software you're using 
    - your computer system 
    - your operating system 
    - description of your problem
