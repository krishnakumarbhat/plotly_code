Glossary
  1. Definition of match: A detection is said to match a re-simulated detection if it has the same RDD range and doppler index(within a scan index), and the difference(error) in the range, range rate, azimuth and elevation are within the thresholds mentioned below
    • Range : 0.01 m
    • Range rate : 0.015 m/s
    • Azimuth : 0.00873 radians
    • Elevation : 0.00873 radians
  2. Accuracy: (Number of matching detections / total number of detections) * 100
  3. Saturation: A SI is said to be saturated if the maximum range among all the vehicle detections is >= 135m and re-simulated detections is < 135m
  4. Plot A: Plot of % of unmatched detections against the corresponding error. Unmatched detections are those detections having same RDD index in both vehicle and re-simulated data but falling outside at-least one of the above mentioned thresholds.
  5. Plot B: Plot of different errors against the corresponding detection's range, range rate, azimuth and elevation
  6. Plot C: Plot of different parameters across scan indices
Note: The plots are interactive
Note: Blank plots indicate no data or zero error
