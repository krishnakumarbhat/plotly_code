Resim Run — How it works

1
Enter paths

Provide the input.txt path and the .simg file path in the form at the top of the Runtime Map page.

2
Auto-detect project

The system extracts the project name from the path (e.g., PLKRA-PROJECTS/RNA-SDV-SRR7/ → project = RNA-SDV-SRR7).

3
Submit via srun

Submits via srun --partition=highPrio --account=<project> running ./rResim_Gen7.sh <input> <simg> highPrio.

4
Track in history

Jobs appear in the resim history panel on the Runtime Map page with status tracking.
